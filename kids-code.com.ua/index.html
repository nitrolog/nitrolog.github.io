Зачем же ребенку отправляться в сферу IT-технологий?


I. Новые возможности

Управлять гаджетами и роботами. Перекладывать решение сложных задач на компьютер. Превращать идеи в виртуальную реальность.

II. Развитие логики и рационального мышления

Обучение программированию развивает логику и позволяет алгоритмизоровать решение любых задач.

III. Разностороннее развитие

Перед начинающим айтишником стоит множество направлений для реализации себя, от программиста до дизайнера, от тестировщика до робототехника.

IV. Огромные перспективы

Ближайшие десятилетия специалисты IT-сферы будут самыми востребоваными во всем мире!

Преимущества формата обучения

IT-школа Kids Code уникальна в своем роде. Каждый из нас глубоко убежден, что информационные технологии это будущее, потому обучать им нужно всех и каждого с раннего детства!
Что делает нашу школу лучшей:

I. Практикующие преподаватели

Учителя, которые действительно могут дать нужные знания и навыки, которые на личном опыте узнали особенности, нюансы, подводные камни и лайфхаки соответствующей сферы

II. Отсутствие территориальной привязки

Школа функционирует на всей территории Киева, потому можно заниматся в удобном для каждого месте и в удобное время

III. Возможность трудоустройства

После окончания курсов лучшие выпускники сразу могут получить работу

Scratch

Scratch разрабатывался как новая учебная среда для обучения школьников программированию. В Скретче можно создавать полноценные фильмы и игры. Это объектно-ориентированная среда, в которой блоки программ собираются из разноцветных кирпичиков, как в конструкторе Лего.

Длительность курса: 18 часов
Возраст: 6+
Требования: Работа с мышкой и клавиатурой

ScratchJr

ScratchJr это вводный язык, который позволяет детям от 5 лет создать свои собственные интерактивные истории и игры. Интерфейс ScratchJr подходит для самых маленьких будущих программистов. Даная среда развивает логику, рациональное мышление и память.

Длительность курса: 18 часов
Возраст: 4+
Требования: Требований нет

Работа в интернете

Компьютер стал неотъемлемой частью нашей жизни. С использованием компьютера сразу решается одна из главнейших, самых острых на сегодня проблем — получение образования, безопасный и равный для всех доступ к полезной и важной информаци.

Длительность курса: 6 часов
Возраст: 6+
Требования: Работа с мышкой и клавиатурой

Знакомство с ПК

Компьютер в наше время - уже больше не малопонятное "чудо техники", а привычная, нужная в работе вещь и удобный образовательный инструмент, просто одно из эффективных средств обучения и развития детей. Это огромная энциклопедия с поражающими воображение возможностями.

Длительность курса: 8 часов
Возраст: 5+
Требования: Требований нет

Азы программирования

В интернете можно встретить много статей о том, как круто быть программистом, как можно много зарабатывать и быть востребоваными. Все это верно, но программирование стоит изучать не только будущим программистам по целому ряду причин.

Длительность курса: 12 часов
Возраст: 12+
Требования: Требований нет

Python

Python подходит для решения львиной доли повседневных задач, будь то резервное копирование, чтение электронной почты, либо же какая-нибудь игрушка. Язык программирования Python практически ничем не ограничен, поэтому также может использоваться в крупных проектах.

Длительность курса: 24 часа
Возраст: 12+
Требования: Базавое пользование ПК

Java

Java – универсальный объектно-ориентированный язык программирования высокого уровня, со строгой типизацией данных и С-подобным синтаксисом. Программы, созданные на Java, транслируются в байт-код, который затем выполняется виртуальной машиной.

Длительность курса: 24 часа
Возраст: 12+
Требования: Уверенное пользование ПК

C/C++

C++ — чрезвычайно мощный язык, содержащий средства создания эффективных программ практически любого назначения, от низкоуровневых утилит и драйверов до сложных программных комплексов самого различного назначения.

Длительность курса: 24 часов
Возраст: 12+
Требования: Уверенное пользование ПК

C#

C# — элегантный, типобезопасный объектно-ориентированный язык, предназначенный для разработки разнообразных безопасных и мощных приложений, выполняемых в среде .NET Framework.

Длительность курса: 24 часов
Возраст: 12+
Требования: Уверенное пользование ПК

Разработка под Android

В наше время больше 80% смартфонов работает на базе Android, потому количество вакансий Android-разработчиков растет как в Украине, так и в мире. Пройдя данны курс дети смогут применить новые знания на практике и начать разрабатывать свои программные продукты.

Длительность курса: 36 часов
Возраст: 12+
Требования: Знания основ программирования

Разработка под iOS

Чтобы программировать вообще и для iOS в частности, надо довольно много знать. Современные технологии избавили программиста от необходимости досконально знать архитектуру компьютера, однако понимать базовые механизмы.

Длительность курса: 36 часов
Возраст: 12+
Требования: Знания основ программирования

HTML + CSS

Курс HTML+CSS разработан не только для начинающих дизайнеров сайтов и верстальщиков. Ведь многие независимо от своей основной специальности желают уметь создавать веб страницу и осваивать грамотную верстку соответственно всем принятым стандартам.

Длительность курса: 18 часов
Возраст: 12+
Требования: Базовые навыки пользования ПК

Java Script

Язык JavaScript на данный момент является самым распространенным средством создания браузерных интерфейсов. Он полностью интегрируется с HTML/CSS а так же поддерживается всеми распространенными браузерами и включен по умолчанию.

Длительность курса: 24 часа
Возраст: 12+
Требования: Знание основ HTML и СSS

.NET Framework

.NET – идея быстрого и высококачественного создания программ и программного обеспечения. Обучение проходит на базе С#, при этом платформа поддерживает большую часть актуальных языков. .NET – эволюционный двигатель по направлению к будущему IT-технологий.

Длительность курса: 20 часов
Возраст: 12+
Требования: Уверенное пользование ПК

PHP

С помощью PHP разработчики создают сайты, имеющие развитые функции, активно взаимодействующие с пользователем, например: Интернет-магазин, веб - каталоги, движки, почтовую рассылку на сайте, новостную ленту, справочники, форумы и т.д.

Длительность курса: 20 часов
Возраст: 14+
Требования: Знание основ HTML, СSS и основ программирования

SQL

Знание SQL необходимо проджект-менеджерам при написания техзадания на разаработку ПО и программистам для управления данными в реляционных базах данных.

Длительность курса: 18 часов
Возраст: 14+
Требования: Знание основ HTML и СSS

Linux + Unix

Многие не замечают, но Linux уже вокруг нас. Все Android устройства работают на Linux, большинства серверов в Интернете также используют эту операционную систему и есть множество других примеров.

Длительность курса: 18 часов
Возраст: 14+
Требования: Уверенное пользование ПК

Windows Server

Windows Server — линейка серверных операционных систем от компании Microsoft. Курс Windows Server предназначены для широкого круга ИТ-специалистов, а также для тех, кто только собирается ими стать.

Длительность курса: 18 часов
Возраст: 14+
Требования: Уверенное пользование ПК

Веб-дизайн

Курс рассчитан на изучение программы Adobe Photoshop и ориентирован на людей, которые планируют заняться веб-дизайном. После прохождения этого курса, Вы будете уметь пользоваться данной программой на необходимом для веб-дизайнера уровне.

Длительность курса: 24 часа
Возраст: 12+
Требования: Уверенное пользование ПК

Основы Photoshop

Курс создан для новичков или тех, кто никогда не пользовался Adobe Photoshop. Он позволит комплексно овладеть программой и научиться как обрабатывать фотографии, так и создавать иллюстрации с нуля.


Длительность курса: 24 часа
Возраст: 12+

Основы Illustrator

Этот курс позволит ученикам всю мощь и гибкость Adobe Illustrator для любых целей. Он подходит как для новичков, так и для тех, кто вообще не работал с этой программой.



Длительность курса: 24 часа
Возраст: 12+
Требования: Уверенное пользование ПК

В этом курсе Вы научитесь монтировать мобильные ролики с помощью PREMIERE PRO - профессиональной программой для нелинейного видеомонтажа.

Длительность курса: 24 часа
Возраст: 12+
